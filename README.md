# Semana Power BI Imparable
![image](https://github.com/lililads/semanapowerbiimparable/assets/159642180/cbc07179-cdd6-4427-84b9-1545bd8371ec)
<p>
La Semana Power BI Imparable fue una serie de 4 clases acerca de los primeros pasos en Power BI que fueron impartidas por Rogelio Salinas en el canal en Youtube Hazlo con Excel. Este evento se llevó cabo del 11 al 16 de junio del presente año.
Los temas vistos junto con el Dashboard producto de cada una fueron:</p>
<ol>
    <li><b>El flujo de trabajo dentro de Power BI - <i>Dashboard Ventas Tech</i></b>
    En esta clase se enseñó lo básico acerca de esta útil herramienta como importar datos (que en este caso para todos los Dashboards fue desde Excel), los tipos de gráficos, qué es Power Query, etc.</li>
    <li><b>¿Qué es DAX y todo el poder dentro de Power BI? - <i>Dashboard Productividad</i></b>
    En esta clase, como su nombre lo indica, se practicó bastante el tema de DAX, la diferencia entre Medidas Implícitas y Explícitas y se profundizó en estas últimas.</li>
    <li><b>Mejores prácticas de diseño, escaneo de datos + objetos personalizados en Power BI. - <i>Dashboard Financiero</i></b>
    El objetivo de esta clase fue mostrar como operan los objetos personalizados. En esta ocasión, el turno fue para Enlighten Data Story.</li>
    <li><b>El Modelo de Datos en Power BI + Normalización de Bases de Datos - <i>Dashboard Resumen Financiero</i></b>
    En esta clase se enseñó sobre el Modelo de Datos en Power BI, como conectar una llave primaria de una tabla con la foránea de otra tabla.</li>
</ol>
<h2>Dashboard 1: Ventas Tech</h2>
<p><b>OBJETIVO:</b> Medir los rendimientos de la empresa.
Para esto se utilizaron distintos objetos como se puede apreciar en la imagen:</p>
<ul><li>Facturación Total, Mejor Producto y Mejor Categoría: <b>Tarjetas</b>.</li>
    <li>Suma de Facturación por Continente: <b>Mapa</b>.</li>
    <li>Suma de Facturación por Marca: <b>Barras agrupadas</b>.</li>
    <li>Suma de Facturación por Año: <b>Anillos</b>.</li>
    <li>Suma de Facturación y Suma de Cantidad Vendida por Año y Mes: <b>Columnas apiladas y Líneas (gráfico combinado)</b>.</li>
</ul>

![image](https://github.com/lililads/semanapowerbiimparable/assets/159642180/7343e02e-e3bf-4d20-a6f9-5cbf8eb350a5)

<h2>Dashboard 2: Productividad</h2>
<p><b>OBJETIVO: </b>Medir productividad: Evaluar que tan eficientemente los recursos como el tiempo y el esfuerzo del operador están siendo utilizados para generar productos de calidad.</p>
<ul><li>Filtro por Operador y Mes: <b>Segmentación de datos</b>.</li>
    <li>Piezas Buenas Producidas, Piezas Rechazadas, Piezas Producidas, Horas Productivas: <b>Tarjetas</b>.</li>
    <li>Horas Productivas vs. Horas no Productivas por Ubicación: <b>Barras agrupadas</b>.</li>
    <li>Piezas buenas producidas por mes: <b>Áreas</b>.</li>
    <li>Porcentajes de Calidad y Productividad: <b>Medidor</b></li>
</ul>

![image](https://github.com/lililads/semanapowerbiimparable/assets/159642180/d2581014-8fc1-4d49-8b97-0007cca59337)

<h2>Dashboard 3: Financiero</h2>
<p><b>OBJETIVO:</b> Analizar distintos movimientos financieros de algunos bancos </p>
<ul><li>Pagos Recibidos, Pagos Hechos, Impuestos, Utilidad: <b>Tarjeta</b></li>
    <li>Utilidad por mes: <b>Cascada</b>.</li>
    <li>Bancos: <b>Segmentación de datos (nueva)</b>.</li>
    <li>Número de movimientos con tarjeta: <b>Enlighten Data Story</b>, Porcentaje de Utilidad: <b>Anillo + Tarjeta</b>.</li>
    <li>Pagos recibidos por banco: <b>Barras agrupadas</b>.</li>
</ul>

<p><b>OBJETIVO:</b> Visualizar el resumen financiero anual de la empresa:</p>
